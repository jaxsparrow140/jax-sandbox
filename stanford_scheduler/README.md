# Stanford Course Scheduler

A web app for searching Stanford courses via natural language and building a schedule with GSB-focused AI/ML recommendations.

## Setup

```bash
pip install -r requirements.txt
python app.py
```

Then open http://localhost:5000 in your browser.

## Features

- Natural language course search via Stanford Explore Courses API
- Curated AI/ML recommendations for GSB students
- Schedule builder with unit tracking
- Export schedule to .ics calendar file
